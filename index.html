CP2407 Information Visualisation

This is a web page intergrated with the function of frequency analysis of text written in JavaScript, which you can pick a text file from your computer and look at the number of occurrence (frequency) of all alphabetical character (ignoring cases) within the context. The graph below will display the number of occurrence once the file is loaded successfully. You are also allow to sort the chart in descending order and clear the graph by pressing the buttons below the graph.
